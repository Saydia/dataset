# 🌾 Rice Leaf Disease Dataset

This repository contains a rice leaf disease image dataset collected under natural field conditions for agricultural computer vision and deep learning research.

## 📌 Dataset Information

- **Total Images:** 4,835
- **Number of Classes:** 7
- **Image Type:** RGB field images
- **Image Condition:** Real-world agricultural environment

The dataset contains variations in:

- Lighting conditions
- Plant age
- Lesion severity
- Imaging angles
- Background complexity

These variations make the dataset suitable for realistic rice disease classification tasks.

---

## 📂 Classes

The dataset includes the following categories:

1. Bacterial Leaf Blight  
2. Brown Spot  
3. Healthy Leaf  
4. Leaf Blast  
5. Leaf Scald  
6. Rice Hispa  
7. Sheath Blight  

---

## 📖 Source

The dataset was collected from the publicly available Rice Leaf Bacterial and Fungal Disease Dataset on Mendeley Data.
